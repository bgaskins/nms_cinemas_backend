NMS Cinemas:

A movie ticket ordering app that allows customers to purchase tickets from anywhere.
<br>
<br>
<br>

Built with:

Java<br>
HTML5<br>
CSS5<br>
JavaScript<br>
Typescript<br>

MySql<br>
Bootstrap<br>
Angular<br>
Spring Boot<br>
